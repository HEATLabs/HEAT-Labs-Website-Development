Website Statistics

Analytics and insights about HEAT Labs traffic and visitor behavior

Website Analytics

Our website statistics are powered by an in-house, privacy-focused tracker that counts page views using dedicated images. It collects no personal data, no IP addresses, cookies, user agents, browser fingerprints, or device types. These fully anonymous view counts help us understand which content is popular and are displayed across the site, including blogs, tank pages, and guides, as helpful indicators. Page View statistics are updated in real time, while CDN statistics are refreshed every 24 hours. Our website also uses Cloudflare Web Analytics that does not use cookies and does not track individual users.

Website view tracking was started on May 31, 2025 and JsDelivr CDN tracking was started on October 5, 2025.

The tracking system is transparent and open source - you can inspect the code by clicking here.
