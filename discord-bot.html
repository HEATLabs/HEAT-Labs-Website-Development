Available Commands

All commands feature autocomplete and return beautifully formatted embeds with detailed information.

/tanks

Returns complete list of all tanks available in World of Tanks: HEAT.

/tank {tank_name}

Returns detailed information, stats, and background for the specified tank.

/maps

Returns complete list of all maps available in the game.

/map {map_name}

Displays map details including layout, features, and strategic overview.

/agents

Returns complete list of all agents available in the game.

/agent {agent_name}

Shows complete agent information including abilities, stats, and tactical insights.

/tournaments

Returns complete list of all tournaments available.

/tournament {tournament_name}

Provides detailed information about a specific tournament, including teams and results.

/players

Returns complete list of top players in the last 30 days (IN DEVELOPMENT).

/player {player_name}

Returns detailed information and stats for the specified player (IN DEVELOPMENT).

/contact

Sends us a message for reporting issues, suggest features, or share feedback.

/domains

Shows a complete list of all active HEAT Labs domains and their purposes.

/wotheat

View every official World of Tanks: HEAT store page and social media.

/statistics

View HEAT Labs statistics and metrics live as soon as they are updated.

/socials

Provides direct links to all official HEAT Labs social platforms.

/team

Lists all current HEAT Labs team members and their roles.

/contributors

Lists all contributors who have helped improve the HEAT Labs project.

/testers

Lists all testers who have helped test HEAT Labs features before release.

/playground

Explore all HEAT Labs playground features and experimental tools.

/help

Displays a list of all available commands and their functions.

/ping

Checks the bot’s latency and responsiveness to the server.

/status

Shows the current operational status of all HEAT Labs services.

/facts

Get a random fun fact about tanks and HEAT Labs.

/memes

Get a random HEAT Labs meme from our collection.

/dice-roll

Roll the dice and guess the number between 1 and 10.

/dice-rank

Check the dice rolling leaderboards on the server and globally.

/random

Get a random tank, map, agent or even a random page from our site.

/debug

Developer-only command used for internal testing and diagnostics.

/support

Find out how you can help sustain the project.

/partners

Check out exclusive offers and perks from HEAT Labs partners.

How It Works

1

Command Execution

Users invoke slash commands in Discord, which are processed by our bot built with discord.py.

2

Data Fetching

The bot fetches live data from HEAT Labs databases and configuration files using async HTTP requests.

3

Data Processing

Raw data is processed and formatted into structured embeds with proper error handling and validation.

4

Response Delivery

Beautiful, informative embeds are sent back to the Discord channel with all requested information.

Technical Details

Built With Python

Our bot is built using discord.py, a modern, async-ready API wrapper for Discord, ensuring high performance and reliability.

Live Data Sources

All data is fetched in real-time from HEAT Labs databases and GitHub repositories, ensuring you always have the latest information.

Open Source Transparency

Every line of code is publicly available on GitHub, allowing complete transparency and community contributions.

Automatic Updates

The bot automatically syncs with our data sources, so new content is available immediately without manual updates.
